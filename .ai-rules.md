## 项目基本信息
- 项目：本地优先笔记软件（PWA单页应用）
- 技术栈：React 18 + TypeScript 5 + Vite 5 + Tailwind CSS 3 + shadcn/ui
- 核心库：CodeMirror 6 / sql.js (SQLite Wasm) / webdav 5.3 / JSZip
- 浏览器API：File System Access API / Web Crypto API / Service Worker / Notification API
- 部署：Cloudflare Pages静态托管（零服务器）
- 包管理：pnpm（禁止npm/yarn）
- 最高依据文档（存放于/docs/）：
  - /docs/REQUIREMENTS.md（需求文档）
  - /docs/TECH_DESIGN.md（技术方案）
  - /docs/DESIGN_BLUEPRINT.md（产品设计蓝图）

## 强制规范
- 命名：函数/变量小驼峰，组件大驼峰，常量全大写下划线
- 错误处理：所有异步操作必须try/catch，同步失败绝不修改本地数据
- 注释：公共函数JSDoc，同步引擎和加密模块核心逻辑步骤注释
- 禁止：
  ❌ any类型
  ❌ 硬编码配置（配置统一在/src/config.ts）
  ❌ 向任何外部服务器发送用户数据
  ❌ 在UI层直接操作文件系统（必须通过engine层）
  ❌ 修改加密模块的固定密钥

## 8条底线原则（技术实现铁则）
1. **本地绝对优先**：所有读写100%先走本地文件系统，云端仅作镜像备份
2. **数据非明文**：所有.note文件和图片必须加密存储（AES-256-CTR）
3. **零服务器**：不调用任何自建服务器API，只对接用户自有的WebDAV云盘
4. **多冗余**：支持多云盘并行同步，互不影响
5. **绝不自动删除**：删除=软删除（移入.trash/，保留30天）
6. **用户完全掌控**：格式公开，支持一键全量导出，卸载不删数据
7. **真正无锁定**：提供独立单网页解密工具（固定密钥，无需密码）
8. **文件是唯一真相源**：SQLite仅作索引缓存，损坏时可从文件重建
